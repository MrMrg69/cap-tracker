# CapTracker

Organize sua lista de leitura de manhuas/manhwas em um app local. Cadastre obras, acompanhe capítulos, marque status, adicione notas/tags e encontre tudo com filtros e busca. Dados ficam armazenados localmente.

## Visao geral
O cap-tracker é um aplicativo local para quem quer manter controle da própria lista de leitura sem depender de serviços externos. O foco é simplicidade: cadastro rapido, acompanhamento de progresso e consulta fácil.

## Recursos
- Cadastro de obras com título, status e observações.
- Acompanhamento de capítulos lidos e por ler.
- Tags e notas para organização pessoal.
- Busca e filtros para localizar rapidamente.
- Armazenamento local (seus dados ficam na sua máquina).

## Como usar
1) Cadastre uma obra.
2) Atualize o progresso conforme for lendo.
3) Use tags/observações para destacar informações importantes.
4) Filtre ou pesquise quando precisar encontrar algo.

## Dados e privacidade
Todos os dados são armazenados localmente. Não há sincronização com serviços externos por padrão.
